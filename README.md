# **📚 國立政治大學學分學程修習檢核工具**

這是一個專為國立政治大學學生設計的工具，透過上傳「全人資料」JSON 檔案，快速檢核其修習進度，並判斷是否滿足特定學分學程或微學程的修畢要求。

## **🌐 線上版本**

本專案已正式佈署，您可以直接訪問以下網址使用：

* **前端介面：** [https://program-checker.onrender.com](https://program-checker.onrender.com)
* **後端 API 服務：** `https://program-checker-backend.onrender.com`

## **✨ 專案特色**

* **自動化檢核：** 擺脫手動對照 PDF 的繁瑣過程，一鍵匯入即可得知修習狀態。  
* **支援多學程複選：** 可同時針對多個感興趣的學程進行進度交叉檢核。  
* **前後端分離架構：** 採用 Go (Golang) 處理核心邏輯，確保計算準確且高效。  
* **響應式設計：** 使用 Vue 3 與 Tailwind CSS，支援手機、平板及電腦瀏覽。  
* **資料隱私：** 檢核過程僅讀取檔案內容，不儲存任何個人學術成績資料。

## **🚀 快速開始 (開發環境)**

如果您想在本地環境運行或參與開發，請參考以下步驟。

### **步驟 1: 後端環境 (Go)**

1. 進入目錄：
```bash
cd backend
```  
2. 安裝依賴：
```bash
go mod tidy
```  
3. 啟動服務(預設運行於 `:10000` 或 `:8080`)：
```bash
go run main.go
```

### **步驟 2: 前端環境 (Vue)**

1. 進入目錄：
```bash
cd frontend  
```
2. 安裝依賴：
```bash
npm install  
```
3. 設定環境變數：在 `.env.local` 設定
```bash
VITE\_API\_BASE\_URL=http://localhost:10000
``` 
4. 啟動開發伺服器：
```bash
npm run dev
```

## **💡 使用指南**

1. **取得資料：** 登入政大全人系統，將個人成績資料匯出為 JSON 格式。  
2. **上傳檢核：** 在本工具中點擊「上傳全人資料」，選擇該 JSON 檔。  
3. **選取目標：** 勾選想要檢核的學分學程（例如：國防安全微學程）。  
4. **查看報告：** 系統將即時列出已通過科目、缺漏科目及總學分數達成率。

## **📝 學程定義維護**

各學程的最新修習要求定義於 `backend/programs.json` 中。若學程規定有更動，只需調整 JSON 結構內的 `min\_credits` 與 `courses` 清單即可完成更新。

## **🔒 隱私與安全聲明**

本工具僅作為輔助參考之用，最終修畢資格仍須以學校教務處之正式審核結果為準。所有上傳的檔案僅在記憶體中進行運算處理，關閉網頁後資料即消失。
